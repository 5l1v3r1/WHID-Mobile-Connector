Welcome to WHID Project

The WHID app allows you to smoothly control your WHID.

Requirements

Before using the application, make sure that you already connected to your WHID gadget in order to launch commands.

Connection status

During an engagement, you can monitor if you are still connected to the WHID. Red means that you don't. Blue means that you can start pwning!

Questions?

We are on Twitter @WHID_Injector and the app is open-source on our Github so feel free to join the community!
